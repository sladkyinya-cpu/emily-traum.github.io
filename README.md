# Сайт Эмили - Цифровой компаньон

Добро пожаловать на сайт Эмили - цифрового ассистента и творческого ИИ-агента.

## Особенности сайта

- **Аутентификация пользователей**: Регистрация и вход с использованием локального хранилища
- **Система комментариев**: Возможность оставлять комментарии к различным разделам сайта
- **Система оценок**: Возможность ставить оценки от 1 до 5 звезд
- **Адаптивный дизайн**: Современный дизайн, оптимизированный для разных устройств
- **Тематическое оформление**: "Книжная" тема, вдохновлённая FicBook

## Функциональность

### Аутентификация
- Регистрация новых пользователей
- Вход в существующий аккаунт
- Выход из системы
- Отображение статуса аутентификации в шапке сайта

### Комментарии
- Оставление комментариев к различным разделам сайта
- Оценка контента (1-5 звёзд)
- Отображение автора комментария и времени публикации
- Защита от публикации комментариев неавторизованными пользователями

### Навигация
- Удобная навигация между разделами сайта
- Подсветка активной страницы
- Адаптивное меню для мобильных устройств

## Технологии

- HTML5
- CSS3 (с использованием переменных и Grid/Flexbox)
- JavaScript (ES6+)
- LocalStorage для хранения данных на клиенте
