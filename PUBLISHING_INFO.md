# Инструкция по публикации сайта

## Варианты публикации

### 1. GitHub Pages (рекомендуется)

1. Создайте репозиторий на GitHub с именем `emily.github.io`
2. Загрузите все файлы из этой папки в репозиторий
3. В настройках репозитория включите GitHub Pages, выбрав ветку `main` и папку `/ (root)`
4. Ваш сайт станет доступен по адресу https://emily.github.io

### 2. Netlify

1. Зарегистрируйтесь на https://netlify.com
2. Перетащите папку с сайтом в панель управления
3. Netlify автоматически развернет сайт
4. Вы получите уникальный URL для доступа к сайту

### 3. Vercel

1. Зарегистрируйтесь на https://vercel.com
2. Используйте CLI или веб-интерфейс для деплоя папки с сайтом
3. Vercel автоматически развернет статический сайт

### 4. Другие хостинги

Любой хостинг, поддерживающий статические сайты, может быть использован для публикации этого сайта.

## Проверка локально

Для локальной проверки сайта вы можете:

1. Использовать Python Simple Server:
   ```
   cd emily-website
   python -m http.server 8000
   ```
   
2. Или использовать Live Server в VS Code
   
3. Или открыть index.html напрямую в браузере (некоторые функции могут не работать без сервера)

## Требования к хостингу

- Поддержка статических файлов (HTML, CSS, JS)
- Поддержка HTTPS
- Хорошая скорость загрузки
